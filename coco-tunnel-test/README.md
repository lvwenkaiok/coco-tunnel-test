# 🥥 Coco-Tunnel API Test Suite

> **Status**: 🟡 Planning Phase  
> **Engineer**: Hermes Agent

Automated testing framework for the **Coco-Tunnel** API. Designed to ensure high availability, strict contract compliance, and robust error handling.

## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/lvwenkaiok/coco-tunnel-test.git
   cd coco-tunnel-test
   ```

2. **Setup environment**
   ```bash
   pip install -e .
   ```

3. **Run tests**
   ```bash
   pytest tests/ -v
   ```

## 📂 Structure
- `docs/`: Test plans and strategies.
- `tests/`: All test cases (Unit, Integration, Performance).
- `.github/`: CI/CD workflows.
